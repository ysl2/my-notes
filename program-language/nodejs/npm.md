# [npm](http://nodejs.cn/learn/an-introduction-to-the-npm-package-manager)

## [查看本地安装的所有npm包](https://zhuanlan.zhihu.com/p/108822110)

> 加上`-g`表示全局。如果只想看当前的，不要加`-g`

查看

```bash
npm list -g --depth 0  // 若需要权限 请加上 sudo 命令
```

更新

```bash
npm update -g xxx
```

删除

```bash
npm uninstall -g xxx
```
